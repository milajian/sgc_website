## ✅ 当前状态
- ✅ ngrok 已安装
- ✅ 前端服务运行中
- ✅ 后端服务运行中
- ⏳ **只需配置 authtoken**

## 🎯 快速配置（3步）

### 步骤1：获取 authtoken（1分钟）

1. **打开浏览器，访问：**
   ```
   https://dashboard.ngrok.com/signup
   ```
   注册账号（如果还没有）

2. **登录后访问：**
   ```
   https://dashboard.ngrok.com/get-started/your-authtoken
   ```
   复制你的 authtoken（类似：`2xxxxxxxxxxxxxxxxxxxxxxxxxxxxx_xxxxxxxxxxxxx`）

### 步骤2：配置 authtoken（30秒）

在终端运行：
```bash
export PATH="$HOME/bin:$PATH"
ngrok config add-authtoken YOUR_AUTHTOKEN
```
（将 `YOUR_AUTHTOKEN` 替换为你刚才复制的 token）

### 步骤3：启动 ngrok（30秒）

运行：
```bash
./一键配置.sh
```

或者直接启动：
```bash
./start-ngrok-local.sh
```

## 📝 配置前端 API（1分钟）

启动 ngrok 后：

1. **查看后端 ngrok 地址**
   - 终端会显示类似：`Forwarding https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:3001`
   - 或访问：http://localhost:4041

2. **更新 `.env.local`**
   ```bash
   # 编辑文件，更新为：
   NEXT_PUBLIC_API_URL=https://你的后端ngrok地址
   ```

3. **重启前端服务**
   - 按 `Ctrl+C` 停止前端
   - 重新运行：`npm run dev`
