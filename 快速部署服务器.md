# 🚀 快速部署 IP Server

## 一键部署

```bash
npm run deploy:server
```

## 部署步骤

脚本会自动完成以下操作：

1. ✅ 检查本地代码和依赖工具
2. 🔌 测试服务器连接
3. 📁 创建服务器目录
4. 📤 上传后端代码
5. 📦 安装依赖
6. 🔄 检查/安装 PM2
7. 🚀 启动/重启服务
8. ✅ 验证服务状态

## 部署后访问

- **健康检查**: http://47.106.73.160:3001/health

## 管理服务

```bash
# SSH 到服务器
ssh root@47.106.73.160

# 查看服务状态
pm2 list

# 查看日志
pm2 logs sgc-backend

# 重启服务
pm2 restart sgc-backend
```
