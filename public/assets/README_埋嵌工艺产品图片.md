# 埋嵌工艺产品图片说明

2. **maiqian2.png** - 埋嵌分立式电容
   - 建议：展示分立式电容埋嵌在 PCB 中的图片

3. **maiqian3.png** - 埋嵌分立式电阻
   - 建议：展示分立式电阻埋嵌在 PCB 中的图片
   - 尺寸：建议 400x400 像素或更高

4. **maiqian4.png** - 散热铜排
   - 建议：展示散热铜排的图片

5. **maiqian5.png** - 埋嵌薄膜式电容
   - 建议：展示薄膜式电容埋嵌在 PCB 中的图片
   - 尺寸：建议 400x400 像素或更高

## 图片要求

- **格式**：PNG（推荐，支持透明背景）或 JPG
- **尺寸**：建议 400x400 像素或更高，保持宽高比一致
- **背景**：透明或白色背景，符合产品展示风格
- **风格**：清晰、专业，突出埋嵌工艺的特点

代码中已使用 `getImagePath()` 函数处理路径，会自动处理 basePath 配置。添加图片文件后，页面会自动显示图片，无需修改代码。
