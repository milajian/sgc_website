# 产品研发方向图片说明

以下8个产品研发方向需要对应的图片文件，请将图片放置在 `public/assets/` 目录下：

1. **ATE 探针卡** - `ate-probe-card.png`
   - 建议：ATE探针卡的特写图片，展示探针结构

2. **软硬结合板** - `rigid-flex-pcb.png`
   - 建议：软硬结合板的实物图片，展示刚性和柔性部分的结合

3. **增材制造** - `additive-manufacturing.png`
   - 建议：3D打印或增材制造过程的图片

4. **PCB线圈** - `pcb-coil.png`
   - 建议：PCB线圈的俯视图或特写，展示线圈图案

5. **玻璃基板** - `glass-substrate.png`
   - 建议：玻璃基板的实物图片，展示透明或半透明的基板

6. **SiC埋嵌** - `sic-embedded.png`
   - 建议：SiC芯片埋嵌在PCB中的图片，展示埋嵌工艺

7. **铜浆烧结** - `copper-paste-sintering.png`
   - 建议：铜浆烧结工艺的图片，展示烧结后的导电路径

8. **陶瓷基板** - `ceramic-substrate.png`
   - 建议：陶瓷基板的实物图片，展示基板表面和结构

## 图片要求

- 格式：PNG（推荐）或 JPG
- 尺寸：建议 400x400 像素或更高
- 背景：透明或白色背景
- 风格：清晰、专业，符合技术产品展示风格

## 当前状态

代码已更新，使用上述文件名。如果图片文件不存在，卡片会显示产品名称作为后备方案。
